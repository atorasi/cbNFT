#cbNFT

software, withdraws funds from OKX and minitizes Coinbase NFT on the Base network


How to use:

- download Golang https://go.dev/doc/install

> fill okx, delay and etc settings in config.yaml

console commands:

- go mod download 

- go run . 

if you want to compile code 

- go build main.go

- ./main
